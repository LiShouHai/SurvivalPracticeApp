# 肩部动作库方案 (按区域划分)

## 1. 区域分类 (按部位划分)
参考胸部与背部的分类，肩部（三角肌）通常划分为三个主要束头，再加上综合/斜方肌区域：
1. **三角肌前束 (Anterior Deltoid)**：负责肩部前侧。
2. **三角肌中束 (Lateral Deltoid)**：决定肩部宽度，是打造“南瓜肩”的关键。
3. **三角肌后束 (Posterior Deltoid)**：负责肩部后侧与立体感。
4. **综合 / 斜方肌 (Comprehensive / Traps)**：推举动作及上背部连接处。

## 2. 基础动作列表 (按区域分类)

### 1) 三角肌前束 (前侧)
- **哑铃推举 (Dumbbell Shoulder Press)** (哑铃) - *基础力量动作*
- **杠铃推举 (Barbell Overhead Press)** (杠铃)
- **哑铃前平举 (Dumbbell Front Raise)** (哑铃)
- **阿诺德推举 (Arnold Press)** (哑铃) - *增加前束拉伸与多维刺激*
- **史密斯推举 (Smith Machine Press)** (固定器械)

### 2) 三角肌中束 (宽度)
- **哑铃侧平举 (Dumbbell Lateral Raise)** (哑铃) - *核心训练动作*
- **绳索侧平举 (Cable Lateral Raise)** (绳索) - *保持持续张力*
- **直立划船 (Upright Row)** (杠铃/绳索)
- **器械侧平举 (Machine Lateral Raise)** (固定器械)

### 3) 三角肌后束 (立体感)
- **俯身哑铃飞鸟 (Bent-over Rear Delt Fly)** (哑铃)
- **蝴蝶机反向飞鸟 (Reverse Machine Fly)** (固定器械)
- **绳索面拉 (Cable Face Pull)** (绳索) - *也是极佳的肩部健康动作*
- **单臂绳索后拉 (Single-arm Cable Rear Delt Pull)** (绳索)

### 4) 综合 / 辅助 (力量与斜方)
- **哑铃耸肩 (Dumbbell Shrug)** (哑铃) - *针对斜方肌上部*
- **地雷管推举 (Landmine Press)** (杠铃/辅助)
- **古巴推举 (Cuban Press)** (哑铃) - *加强肩袖肌群*
