# uni-app 小程序多语言方案说明

## 1. 目标

在 uni-app（Vue3 架构）小程序中实现：

- 支持多语言切换（如 zh-CN / en-US）
- 支持运行时动态切换
- 支持语言持久化存储
- 兼顾主包体积控制
- 可扩展至多单位 / 多币种 / 多地区

适用于：MVP 快速上线阶段 + 后续商业化扩展。

------

# 2. 技术选型

推荐方案：

```
vue-i18n@9 + Composition API
```

原因：

- 官方生态成熟
- Vue3 原生支持
- 支持响应式切换
- 支持懒加载语言包
- 可扩展性强

不推荐：

- 手写 JSON + 全局变量（不可维护）
- 后端翻译接口（增加复杂度）
- 纯字符串替换方案（不可扩展）

------

# 3. 项目结构建议

```
src/
 ├─ locale/
 │   ├─ index.ts
 │   ├─ zh-CN.ts
 │   └─ en-US.ts
 ├─ store/
 └─ main.ts
```

# 4. 实现步骤

## 4.1 安装

```
npm install vue-i18n@9
```

------

## 4.2 语言文件示例

### zh-CN.ts

```
export default {
  common: {
    confirm: "确定",
    cancel: "取消"
  },
  home: {
    title: "首页"
  }
}
```

### en-US.ts

```
export default {
  common: {
    confirm: "Confirm",
    cancel: "Cancel"
  },
  home: {
    title: "Home"
  }
}
```

------

## 4.3 创建 i18n 实例

```
import { createI18n } from 'vue-i18n'
import zhCN from './zh-CN'
import enUS from './en-US'

const i18n = createI18n({
  legacy: false,
  locale: uni.getStorageSync('lang') || 'zh-CN',
  fallbackLocale: 'zh-CN',
  messages: {
    'zh-CN': zhCN,
    'en-US': enUS
  }
})

export default i18n
```

------

## 4.4 注册到应用

```
import { createSSRApp } from 'vue'
import App from './App.vue'
import i18n from './locale'

export function createApp() {
  const app = createSSRApp(App)
  app.use(i18n)
  return { app }
}
```

------

## 4.5 页面使用

```
<script setup>
import { useI18n } from 'vue-i18n'
const { t } = useI18n()
</script>

<template>
  <view>
    {{ t('home.title') }}
  </view>
</template>
```

------

# 5. 语言切换实现

```
import i18n from '@/locale'

export function changeLang(lang: string) {
  i18n.global.locale.value = lang
  uni.setStorageSync('lang', lang)
}
```

特点：

- 响应式更新
- 持久化存储
- 页面自动刷新

------

# 6. 小程序特殊处理

## 6.1 tabBar 多语言

小程序 tabBar 不支持模板语法，必须手动设置：

```
uni.setTabBarItem({
  index: 0,
  text: i18n.global.t('home.title')
})
```

语言切换后需要重新调用。

------

## 6.2 禁止写死文本

错误写法：

```
<text>确定</text>
```

正确写法：

```
<text>{{ t('common.confirm') }}</text>
```

------

# 7. 性能优化（语言较多时）

使用动态加载：

```
const messages = {
  'zh-CN': () => import('./zh-CN'),
  'en-US': () => import('./en-US')
}
```

优点：

- 减少主包体积
- 支持按需加载
- 更适合多语言版本

------

# 8. 进阶扩展建议（健身类小程序适用）

建议提前抽象：

### 单位

```
unit.weight
unit.distance
```

### 时间格式

```
format.date
format.time
```

### 币种

```
currency.symbol
```

避免在业务代码中写死：

- kg / lb
- ¥ / $
- 24h / 12h

# 9. 推荐实践总结

| 阶段       | 推荐方案               |
| ---------- | ---------------------- |
| MVP        | 静态引入语言包         |
| 多语言版本 | 动态 import            |
| 多地区运营 | 抽象单位 / 时间 / 币种 |
| 小程序主端 | 手动控制 tabBar        |

# 10. 结论

在 uni-app + Vue3 小程序项目中：

> vue-i18n 是当前最稳定、可维护性最高、扩展能力最强的多语言解决方案。

该方案适合：

- 快速上线
- 后续功能迭代
- 商业化运营
- 多国家扩展
