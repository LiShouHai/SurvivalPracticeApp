# 勋章墙功能设计方案 (Revised)

## 1. 概述
本项目旨在开发一个高度视觉化的“勋章墙”系统，其核心交互参考苹果健康（Fitness）中的奖牌功能。通过 **CSS 3D 渲染** 实现勋章的厚度感与光影效果，配合触感反馈（震动）提升交互体验。

## 2. 页面结构设计

根据需求，共涉及 3 个核心页面：

### 2.1 勋章主墙 (Badge Wall) - `src/pages/badge/index.vue`
*   **布局逻辑**：
    *   **上半部分 (2/3 区域)**：展示最新获得的勋章。使用左右滑动的组件（如 `swiper` 或 `scroll-view`），背景采用渐变或深色磨砂质感。
    *   **下半部分 (1/3 区域)**：展示小勋章列表。作为进入全量列表的入口。
*   **交互逻辑**：
    *   点击顶部大勋章 -> 跳转至 **勋章详情页**。
    *   点击底部列表 -> 跳转至 **勋章列表页**。

### 2.2 勋章列表页 (Badge List) - `src/pages/badge/list.vue`
*   **内容**：网格状排列用户获得的所有勋章。
*   **交互**：点击任意勋章 -> 跳转至 **勋章详情页**。

### 2.3 勋章详情页 (Badge Detail) - `src/pages/badge/detail.vue`
*   **视觉核心**：全屏展示单枚勋章。
*   **3D 实现**：
    *   **厚度感**：通过 CSS 多层叠加与 `box-shadow` 模拟金属边缘，使勋章看起来有实质感而非扁平。
    *   **旋转交互**：按住勋章可自由旋转或通过点击翻转。使用 `vibrateShort` 提供金属撞击/阻尼感的震动反馈。
*   **信息展示**：
    *   **正面**：勋章图形及名称。
    *   **背面**：金属微雕质感，展示：**[用户名]**、**[获得时间]**。
*   **简介区域**：位于 3D 勋章下方，介绍勋章背景。

## 3. 技术方案与交互规范

### 3.1 3D CSS 模拟
由于无 3D 模型资源，我们将使用纯 CSS 构建“立体勋章”：
- **正面层 (Front Layer)**: 渐变色 + 图标。
- **中间层 (Thickness Layers)**: 通过 5-10 层极其微小的 `translateZ` 偏移，形成视觉上的厚度边框。
- **背面层 (Back Layer)**: 展示雕刻文字。

### 3.2 动效与触感反馈 (Vibration)
- **震动反馈**：在翻转勋章切换到 180 度或旋转至边缘位置时，触发 `uni.vibrateShort({ type: 'medium' })`，模拟物理惯性。
- **流畅性**：利用 `requestAnimationFrame` 或 CSS `transition` 确保 3D 转换达到 60FPS。

### 3.3 视觉风格 (UIXPROMAX 规范)
遵循项目已有的 UIXPROMAX 设计准则：
- 采用 **暗黑拟物态 (Dark Skeuomorphism)**。
- 勋章金属材质：金色（#FFD700）、银色（#C0C0C0）、青铜色（#CD7F32）渐变叠加。
- 动态光泽：利用 `radial-gradient` 模拟光源随视角转动的反射效果。

## 4. 路由定义 (Pages)
- `pages/badge/index`: 勋章墙
- `pages/badge/list`: 勋章列表
- `pages/badge/detail`: 勋章详情
