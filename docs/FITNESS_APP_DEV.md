# 健身小程序开发文档

> 基于 UniApp 的微信小程序，包含登录、打卡、训练记录等功能

---

## 1. 项目概述

### 1.1 项目简介

一款简约风格的健身记录小程序，帮助用户：
- 每日打卡签到，记录健身位置
- 记录训练动作、组数、次数、重量
- 通过成就徽章系统激励持续健身

### 1.2 设计风格

- **简约现代**：大量留白，清晰的视觉层次
- **动画流畅**：页面转场、按钮反馈、数据变化均有流畅动画
- **色彩方案**：主色调活力橙 `#FF6B35` + 深灰/纯白背景

### 1.3 技术栈

| 类别 | 技术选型 |
|------|----------|
| 框架 | UniApp (Vue3) |
| UI 组件 | uv-ui（推荐）/ uview-plus / Wot Design Uni |
| 状态管理 | Pinia |
| 数据存储 | 微信本地存储 |
| 动画 | CSS Animation + Lottie |

### 1.4 UI 组件库选型

> ⚠️ **注意**：Vant4 不适用于 UniApp 小程序开发。Vant 的 Web 版使用了 DOM 操作，小程序不支持；Vant-weapp 版本不支持 Vue3 且多端兼容性差。

**推荐组件库对比：**

| 组件库 | GitHub Star | 组件数 | 特点 | 文档地址 |
|--------|-------------|--------|------|----------|
| **uv-ui** ⭐ | 568+ | 70+ | 基于 uView 的 Vue3 版本，热度最高，兼容全端 | [uvui.cn](https://www.uvui.cn/) |
| **uview-plus** | 362+ | 120+ | 基于 uView2.x，组件最丰富，支持鸿蒙/多语言 | [uview-plus.jiangruyi.com](https://uview-plus.jiangruyi.com/) |
| **Wot Design Uni** | 492+ | 70+ | 京东风格，设计精美，TS 支持好 | [wot-design-uni](https://wot-design-uni.netlify.app/) |
| **uni-ui** | 官方 | 60+ | 官方组件库，全端兼容最好，样式简约 | [uniapp.dcloud.net.cn](https://uniapp.dcloud.net.cn/component/uniui/uni-ui.html) |
| **TuniaoUI** | 192+ | 60+ | 图鸟系，设计美观，动画丰富 | [tuniaokj.com](https://vue3.tuniaokj.com/) |
| **TMUI** | - | 100+ | 支持 Vue3/nvue，主题定制强 | [tmui.design](https://tmui.design/) |
| **FirstUI** | - | 200+ | 组件丰富，部分收费 | [firstui.cn](https://www.firstui.cn/) |

**本项目推荐：uv-ui**

选择理由：
- ✅ 基于 uView，社区活跃，文档完善
- ✅ 全端兼容（微信/支付宝/H5/App）
- ✅ 组件丰富，满足常见需求
- ✅ 支持 Vue3，持续更新维护
- ✅ MIT 开源协议，免费商用

**安装方式：**
```bash
# 方式1：通过 npm 安装
npm install @climblee/uv-ui

# 方式2：通过 HBuilderX 插件市场导入
# 插件市场搜索 "uv-ui" 直接导入
```

**配置方式：**
```javascript
// main.js
import uvUI from '@climblee/uv-ui'
app.use(uvUI)

// pages.json 配置 easycom
{
  "easycom": {
    "autoscan": true,
    "custom": {
```

#### 3.1.3 登录流程

**微信授权登录流程：**
```
用户点击微信登录 → 检查协议勾选 → wx.login获取code
→ wx.getUserProfile获取头像昵称 → 保存用户信息 → 跳转首页
```

**账号密码登录流程：**
```
输入手机号/用户名 → 输入密码 → 检查协议勾选
→ 验证格式 → 调用登录接口(预留) → 保存用户信息 → 跳转首页
```

### 3.2 打卡模块

#### 3.2.1 功能清单

| 功能项 | 说明 | 优先级 |
|--------|------|--------|
| 每日打卡 | 每天限打卡1次，点击按钮完成打卡 | P0 |
| 打卡动画 | 打卡成功显示撒花/庆祝动画 | P0 |
| 位置记录 | 获取并记录打卡时的地理位置 | P0 |
| 位置展示 | 显示打卡地点名称（如：XX健身房） | P0 |
| 连续天数 | 计算并显示连续打卡天数 | P0 |
| 最长连续 | 记录历史最长连续打卡天数 | P0 |
| 总打卡数 | 累计打卡总次数 | P0 |
| 打卡日历 | 日历视图展示当月打卡情况 | P0 |
| 打卡详情 | 点击日历日期查看当日打卡详情 | P1 |
| 打卡提醒 | 每日定时推送打卡提醒（需订阅消息） | P2 |
| 补打卡 | 支持补签昨天的打卡（可选，需消耗积分） | P2 |

#### 3.2.2 打卡规则

| 规则 | 说明 |
|------|------|
| 打卡时间 | 每天 00:00:00 - 23:59:59 |
| 打卡次数 | 每天限打卡 1 次 |
| 连续判定 | 相邻日期都有打卡记录即为连续 |
| 中断处理 | 中断一天，连续天数重置为 0 |
| 位置授权 | 用户可拒绝，拒绝后不记录位置 |

## 6. 动画设计

### 6.1 关键动画场景

| 场景 | 动画类型 | 时长 | 说明 |
|------|----------|------|------|
| 打卡成功 | Lottie撒花 | 2s | 撒花/烟火庆祝效果 |
| 徽章解锁 | 缩放+旋转+光效 | 1.5s | 徽章从小到大，闪光 |
| 按钮点击 | 缩放+涟漪 | 0.3s | 按下缩小，涟漪扩散 |
| 页面转场 | 滑动+渐变 | 0.3s | 从右侧滑入 |
| 数字变化 | 滚动数字 | 0.5s | 数字逐位滚动变化 |
| 列表项 | 依次渐入 | 0.3s | 列表项依次从下方滑入 |
| 进度条 | 宽度动画 | 0.5s | 进度条平滑增长 |
| 卡片展开 | 高度动画 | 0.3s | 平滑展开收起 |
| 下拉刷新 | Lottie动画 | - | 自定义加载动画 |
| 空状态 | Lottie动画 | - | 空数据时的提示动画 |

### 6.2 稀有度光效

| 稀有度 | 效果 |
|--------|------|
| 普通 | 无特效 |
| 稀有 | 轻微蓝色光晕 |
| 史诗 | 明显紫色光晕 |
| 传说 | 金色呼吸光效（循环动画） |
